# KAFKA
![imagen](https://github.com/bernabeudario/KAFKA/blob/main/img-portada.jpg?raw=true)

Este Docker-Compose despliega una arquitecura que se compone de:
* Clúster Kafka con tres Servers, configurados en Modo Mixto.
* Interfaz web ligera (https://github.com/provectus/kafka-ui), que utilizaremos para ver los resultados de cada operación.
* Confluent Schema Registry, para almacenar y recuperar esquemas Avro, JSON Schema y Protobuf (https://github.com/confluentinc/schema-registry).

El funcionamiento y los pasos de instalación se encuentran detallados en el siguiente enlace:
* [Abrir enlace Medium...](https://medium.com/@dariobernabeu/apache-kafka-101-guia-conceptual-ac17a80f4a75)
